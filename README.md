## Requirement
- Paddle
- Paddle_OCR
- translators

## Quick Start

> recommend environment is `anaconnda`

- create `paddle_env `

```shell
conda create -n paddle_env python=3.x #use u want version
```

- active environment

```
conda activate paddle_env
```

- install another environment

```
pip install -r requirements.txt
```

- run script(in windows)

```
.\prepare.ps1
```
